# SIKAS
SIKAS (Sistem Informasi Stok dan Kasir) adalah aplikasi berbasis web yang dirancang untuk membantu usaha kecil dan menengah dalam mengelola stok barang dan proses penjualan secara efisien. Aplikasi ini menyediakan fitur-fitur seperti manajemen inventaris, pencatatan transaksi penjualan, laporan keuangan, dan analisis data penjualan.
